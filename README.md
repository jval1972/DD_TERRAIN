# Terrain Generator (DD_TERRAIN)
Terrain Generator is a simple terrain editor.

## Features
Export RAD (Radix game remake) maps 

Export UDMF GZdoom/K8Vavoom maps

Export screenshots to disk

Export screenshots to clipboard

Multiple Undo/Redo
